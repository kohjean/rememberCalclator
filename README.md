# リメンバー・カリキュレーター
## 仕様
* 四則演算に対応。掛け算、割り算を優先 -> 1+2*3=7  
* 累乗非対応、浮動小数点未調整  
* 割る０,最初に演算子から入力 -> エラー
* ユーザー登録機能実装  
* 計算結果の保存機能実装
* 保存した計算結果の表示機能実装
* 保存項目名の作成機能実装
## 使用前に 
* データベース作成必須 -> calc.sqlをインポートしてください
* config.php内、接続情報書き換え必須
## テストデータについて
* テストユーザーが最初から入っています  
* ユーザー名:'それいけかけふ' パスワード: 'root'
